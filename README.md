# JogoMusical
Repositório pro projeto final 

# Integrantes 

Renato Luiz

# Descrição
O jogo é um projeto para introduzir crianças nao mundo da musica, de forma leve e divertida

# Gênero 
Educacional, Plataforma, SinglePlayer

# Público
Crianças de 5 à 11 anos, pais ou professores que querem introduzifr seus filhos ou alunos ao mundo da música

# Tecnologias
Engine da Unity e VS Studio para programação

# Instruções
Para abrir e rodar o projeto na unity, apenas precisa colocar as cenas na build profile, e para abrir no itch io, é só clicar no link do class
